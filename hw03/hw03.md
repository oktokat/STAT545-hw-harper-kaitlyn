Get the maximum and minimum of GDP per capita for all continents.
-----------------------------------------------------------------

``` r
min_max = gapminder %>% 
  group_by(continent) %>% 
  summarise(maxGDP = max(gdpPercap), minGDP = min(gdpPercap))

knitr::kable(min_max, col.names = c("Continent", "Max GDP", "Min GDP"))
```

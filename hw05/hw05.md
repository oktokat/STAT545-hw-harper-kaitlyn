### Reorder the levels of `idno` based on `zlen` variable

``` r
low_zlen_factor = fct_reorder(stunted$idno, stunted$zlen, min) %>% 
  levels() %>% 
  head()

lowest = stunted %>% 
  filter(idno %in% low_zlen_factor) %>% 
  group_by(idno) %>% 
  filter(zlen == min(zlen)) %>% 
  select(idno, zlen)

kable(lowest)
```
